Assignment Overview:
For your final project, you will develop a C++ program based on the business idea you proposed in Week 3. This is your opportunity to apply the concepts and skills you've learned throughout the course in a comprehensive project.

Instructions:
Develop Your Program:
Based on your approved proposal, develop a C++ program that addresses your business idea.
Your program should incorporate the following concepts:
Class Design: Create classes that model real-world entities in your business idea.
Inheritance: Use inheritance to extend your classes with additional functionality.
Arrays and Dynamic Memory: Implement arrays or dynamic memory to manage collections of data.
Functions: Write functions that perform specific tasks within your program.
User I/O: Include user input/output operations interact with a user
Write a Brief Report:
Include a 2-3 page report that describes:
Your business idea and how your program addresses it.
The key classes and functions in your program.
Any challenges you faced and how you overcame them.
Future enhancements you would make if you had more time.
Optional Presentation:
You may choose to present your project to the class. This is optional but recommended, as it allows you to showcase your work and receive feedback from peers.
Final Submission:
Submit your completed program along with the report and optional presentation to the Blackboard assignment.
What to Submit:

A copy of your source code for the project.
A brief report detailing your project.
(Optional) A presentation file if you choose to present your project.
Submission Guidelines:

Ensure your code is well-commented and easy to read.
Name your source code file appropriately based on your project.
Submit your project files, report, and optional presentation to the Blackboard assignment.
